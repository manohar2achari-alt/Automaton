Azure Secret Monitor

Automated Security Hygiene for IAM Teams
1

Create Automation Account

Create a new Automation Account in the Azure Portal.

  • Identity: Enable System Assigned Managed Identity.
  • Network: Set to Public Access (or your specific private config).
2

Import Modules (Runtime 7.2)

This solution uses the modern Graph SDK which requires the latest PowerShell runtime.

Important: When adding modules in Azure Automation, select Runtime version: 7.2 (or 7.1). Do not use 5.1.

Go to Shared Resources > Modules > Add a module and add:

  1. Microsoft.Graph.Authentication
  2. Microsoft.Graph.Applications
3

Grant Permissions

Run this script locally to grant the Managed Identity permission to read apps and send emails.

One-Time Setup Script (Local PowerShell) 
Connect-MgGraph -Scopes "AppRoleAssignment.ReadWrite.All","Application.Read.All"

$AutoAccountName = "aa-secret-monitor-prod" 
$MI = Get-MgServicePrincipal -Filter "displayName eq '$AutoAccountName'"
$Graph = Get-MgServicePrincipal -Filter "appId eq '00000003-0000-0000-c000-000000000000'"

$Roles = @("Application.Read.All", "Mail.Send")

foreach ($RoleName in $Roles) {
    $Role = $Graph.AppRoles | Where-Object {$_.Value -eq $RoleName}
    New-MgServicePrincipalAppRoleAssignedTo -ServicePrincipalId $MI.Id -PrincipalId $MI.Id -ResourceId $Graph.Id -AppRoleId $Role.Id
    Write-Host "Assigned $RoleName" -ForegroundColor Green
}
4

The Runbook (Individual Alerts)

Create a Runbook using PowerShell 7.2. This script triggers one email per expiring secret.

Runbook Source Code 
# --- CONFIGURATION ---
$NotifyDays     = @(60, 30, 7)
$SenderEmail    = "alerts@yourdomain.com" 
$RecipientEmail = "iam-team@yourdomain.com"

try {
    # --- AUTHENTICATION ---
    Connect-AzAccount -Identity
    $token = Get-AzAccessToken -ResourceUrl "https://graph.microsoft.com"
    Connect-MgGraph -AccessToken $token.Token

    $Today = Get-Date
    Write-Output "Scanning App Registrations..."
    $Apps = Get-MgApplication -All -Property DisplayName, AppId, PasswordCredentials

    foreach ($App in $Apps) {
        foreach ($Cred in $App.PasswordCredentials) {
            $Expiry = Get-Date $Cred.EndDateTime
            $DaysLeft = ($Expiry - $Today).Days

            if ($NotifyDays -contains $DaysLeft) {
                
                # --- SEND EMAIL IMMEDIATELY ---
                $AppName = $App.DisplayName
                $AppId   = $App.AppId
                $ExpDate = $Expiry.ToString("yyyy-MM-dd")
                
                $HtmlBody = "
                <h3>⚠️ Action Required: Secret Expiring</h3>
                <p>The secret for the following application is expiring in <strong>$DaysLeft days</strong>.</p>
                <ul>
                    <li><strong>App Name:</strong> $AppName</li>
                    <li><strong>App ID:</strong> $AppId</li>
                    <li><strong>Expiry Date:</strong> $ExpDate</li>
                </ul>
                <p>Please rotate this secret immediately.</p>"

                $Params = @{
                    Message = @{
                        Subject = "ALERT: Secret for '$AppName' expires in $DaysLeft days"
                        Body = @{ ContentType = "HTML"; Content = $HtmlBody }
                        ToRecipients = @( @{ EmailAddress = @{ Address = $RecipientEmail } } )
                    }
                    SaveToSentItems = "false"
                }
                
                Send-MgUserMail -UserId $SenderEmail -BodyParameter $Params
                Write-Output "Sent alert for $AppName ($DaysLeft days left)"
            }
        }
    }
} catch {
    Write-Error "Runbook failed: $_"
}
