Azure Client Secret Expiration Notifications

Overview

This solution monitors Azure App Registration client secrets and sends email notifications 60, 30, and 7 days before expiration. There is no automatic rotation — all notifications go to the IAM distribution list via a dedicated user mailbox.

How It Works

  • Automation Account runbook runs daily
  • Uses Managed Identity with Graph API permissions
  • Detects secrets expiring in 60 / 30 / 7 days
  • Sends email notifications to the IAM distribution list

[Automation Account MI] ---reads---> [App Registrations]
                                \
                                 ---> Sends email notification to IAM DL

Step-by-Step Guide

1. Create Azure Automation Account

Create a PowerShell 7 Automation Account with System-assigned Managed Identity.

2. Assign Managed Identity Permissions

Grant Graph API roles:

  • Application.Read.All
  • Directory.Read.All

3. Configure Notification Mailbox

Create a dedicated mailbox to send notifications to the IAM distribution list. Store credentials securely in Azure Automation as a credential asset.

4. Schedule Runbook

Runbook should execute daily and send notifications for secrets expiring in 60 / 30 / 7 days.

# Parameters
$NotifyDays = @(60,30,7)
$SmtpServer = "smtp.office365.com"
$SmtpPort = 587
$Sender = "automation@yourdomain.com"
$CredentialAssetName = "AutomationMailboxCredential"

# Get credentials from Automation Account
$Cred = Get-AutomationPSCredential -Name $CredentialAssetName

# Acquire Graph token
$token = Invoke-RestMethod -Method Get -Uri "http://169.254.169.254/metadata/identity/oauth2/token?api-version=2018-02-01&resource=https://graph.microsoft.com/" -Headers @{Metadata="true"}
$headers = @{ Authorization = "Bearer $($token.access_token)" }

# Get all applications
$applications = Invoke-RestMethod -Uri "https://graph.microsoft.com/v1.0/applications?$select=id,displayName,passwordCredentials" -Headers $headers

foreach ($app in $applications.value) {
    foreach ($cred in $app.passwordCredentials) {
        if (-not $cred.endDateTime) { continue }
        $expiry = [datetime]$cred.endDateTime
        $daysLeft = [int]([math]::Floor(($expiry - (Get-Date)).TotalDays))
        if ($NotifyDays -contains $daysLeft -or $daysLeft -lt 0) {
            # Compose email
            $subject = "Azure Client Secret Expiration Notification: $($app.displayName)"
            $body = "App: $($app.displayName)`nSecret KeyId: $($cred.keyId)`nExpiry Date: $expiry`nDays Remaining: $daysLeft`nPlease update this secret manually."
            Send-MailMessage -From $Sender -To "IAM_DL@yourdomain.com" -Subject $subject -Body $body -SmtpServer $SmtpServer -Port $SmtpPort -Credential $Cred -UseSsl
            Write-Output "Notification sent for $($app.displayName) expiring in $daysLeft days."
        }
    }
}
