Create Automation Account

We need a container to run our PowerShell script. This is serverless and costs pennies per month.

  • Go to Azure Portal > Automation Accounts > Create.
  • Identity: During creation, enable System Assigned Managed Identity.
  • Network: Set to Public Access (unless you have a specific Private Endpoint requirement).
2

Import Required Modules

The standard Azure Automation environment does not come with the Graph SDK installed. You must add them, or the script will fail.

Action Required: In your Automation Account, go to Shared Resources > Modules > Add a module.

Add these two modules (select Runtime version 7.2):

  1. Microsoft.Graph.Authentication
  2. Microsoft.Graph.Applications
3

Grant Graph Permissions

The Managed Identity needs permission to read all apps and send emails. This cannot be done in the Azure Portal GUI. You must run this script once from your local machine as a Global Admin.

One-Time Setup Script (Local PowerShell) 
# 1. Connect to Graph as Admin
Connect-MgGraph -Scopes "AppRoleAssignment.ReadWrite.All","Application.Read.All"

# 2. CONFIG: Enter your Automation Account Name
$AutoAccountName = "aa-secret-monitor-prod" 

# 3. Get Service Principals
$MI = Get-MgServicePrincipal -Filter "displayName eq '$AutoAccountName'"
$Graph = Get-MgServicePrincipal -Filter "appId eq '00000003-0000-0000-c000-000000000000'"

# 4. Assign Permissions (Read Apps & Send Mail)
$Roles = @("Application.Read.All", "Mail.Send")

foreach ($RoleName in $Roles) {
    $Role = $Graph.AppRoles | Where-Object {$_.Value -eq $RoleName}
    
    New-MgServicePrincipalAppRoleAssignedTo `
        -ServicePrincipalId $MI.Id `
        -PrincipalId $MI.Id `
        -ResourceId $Graph.Id `
        -AppRoleId $Role.Id
        
    Write-Host "SUCCESS: Assigned $RoleName" -ForegroundColor Green
}

The Automation Runbook

Create a new Runbook (select PowerShell 7.2). Paste the script below. This logic scans for secrets expiring in 60, 30, and 7 days and sends a single summary table.

Runbook Source Code 
# --- CONFIGURATION ---
$NotifyDays     = @(60, 30, 7)
$SenderEmail    = "alerts@yourdomain.com"  # Must be a mailbox the Identity has Send-As rights on
$RecipientEmail = "iam-team@yourdomain.com"

try {
    # --- AUTHENTICATION (Managed Identity) ---
    Connect-AzAccount -Identity
    $token = Get-AzAccessToken -ResourceUrl "https://graph.microsoft.com"
    Connect-MgGraph -AccessToken $token.Token

    $ExpiringApps = @()
    $Today = Get-Date

    # --- SCANNING ---
    Write-Output "Scanning App Registrations..."
    $Apps = Get-MgApplication -All -Property DisplayName, AppId, PasswordCredentials

    foreach ($App in $Apps) {
        foreach ($Cred in $App.PasswordCredentials) {
            $Expiry = Get-Date $Cred.EndDateTime
            $DaysLeft = ($Expiry - $Today).Days

            if ($NotifyDays -contains $DaysLeft) {
                $ExpiringApps += [PSCustomObject]@{
                    Name       = $App.DisplayName
                    AppId      = $App.AppId
                    DaysLeft   = $DaysLeft
                    ExpiryDate = $Expiry.ToString("yyyy-MM-dd")
                }
            }
        }
    }

    # --- REPORTING ---
    if ($ExpiringApps.Count -gt 0) {
        $Style = "border: 1px solid #ddd; padding: 8px;"
        $Rows = ""
        
        foreach ($Row in $ExpiringApps) {
            $Rows += "<tr><td style='$Style'>$($Row.Name)</td><td style='$Style'>$($Row.DaysLeft)</td><td style='$Style'>$($Row.ExpiryDate)</td></tr>"
        }

        $Body = "
        <h3 style='font-family:sans-serif;'>⚠️ App Secrets Expiring</h3>
        <table style='border-collapse:collapse; width:100%; font-family:sans-serif;'>
            <tr style='background-color:#f2f2f2;'>
                <th style='$Style'>App Name</th>
                <th style='$Style'>Days Left</th>
                <th style='$Style'>Expiry Date</th>
            </tr>
            $Rows
        </table>"

        $Params = @{
            Message = @{
                Subject = "Action Required: $($ExpiringApps.Count) Apps Expiring"
                Body = @{ ContentType = "HTML"; Content = $Body }
                ToRecipients = @( @{ EmailAddress = @{ Address = $RecipientEmail } } )
            }
            SaveToSentItems = "false"
        }
        
        Send-MgUserMail -UserId $SenderEmail -BodyParameter $Params
        Write-Output "Email sent successfully."
    } else {
        Write-Output "No secrets expiring within the 60, 30, or 7 day window."
    }

} catch {
    Write-Error "Runbook failed: $_"
}
